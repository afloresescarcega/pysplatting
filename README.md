# pysplatting

pysplatting is a Python project aimed at optimizing the reconstruction of a scene through pixelwise operations.

## Installation

To set up the pysplatting environment, follow these steps:

1. Ensure that you have Python installed on your system.
2. Clone the repository to your local machine.
3. Install the required dependencies by running `poetry install`.

## Usage

To run the pysplatting project, you need to provide the path of the image you want to process. Use the following command:

```sh
python main.py path/to/your/image.png
```

Make sure to replace `path/to/your/image.png` with the actual path to the image file you want to use.

## Dependencies

The project relies on the Python Imaging Library (PIL), which is specified in the `pyproject.toml` file.
